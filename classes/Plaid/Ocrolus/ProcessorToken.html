Creates a Ocrolus processor token from an access_token.

Does a POST /processor/ocrolus/processor_token/create call which can be used to generate a Ocrolus processor token for a given account ID.

access_token

access_token to create a public token for.

account_id

ID of the account to create a processor token for.

Returns

Returns a ProcessorTokenResponse object containing a Ocrolus processor token.

def create(access_token, account_id)
  post_with_auth 'processor/ocrolus/processor_token/create',
                 ProcessorTokenResponse,
                 access_token: access_token,
                 account_id: account_id
end
